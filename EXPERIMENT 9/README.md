# Data Visualization II

##  Problem Statement
This project demonstrates **Data Visualization using Python** by analyzing the Titanic dataset.  
1. Use the inbuilt dataset 'titanic' as used in the above problem. Plot a box plot for distribution
of age with respect to each gender along with the information about whether they survived
or not. (Column names : 'sex' and 'age')
2. Write observations on the inference from the above statistics.
---

##  Objectives
- Load and explore the Titanic dataset  
- Handle missing values  
- Visualize age distribution using a box plot  
- Analyze patterns based on gender and survival  

---

##  Technologies Used
- Python  
- Pandas  
- Seaborn  
- Matplotlib  
- Jupyter Notebook  

---

##  Dataset
- Built-in dataset from Seaborn: `titanic`  
- Key columns used:
  - `age`
  - `sex`
  - `survived`

---

##  Steps / Algorithm
1. Import required libraries (pandas, seaborn, matplotlib)  
2. Load dataset using seaborn  
3. Explore dataset and check missing values  
4. Select relevant columns  
5. Handle missing values (drop null values in age)  
6. Add labels and title
7. Display plot
8. Analyze result
6. Create box plot
7. Add labels and title
8. Display plot
9. Analyze result

## Observations
 - Females have a higher survival rate than males
 - Younger passengers had better chances of survival
 - Males show more non-survivors, especially in higher age groups
 - Outliers represent elderly passengers
